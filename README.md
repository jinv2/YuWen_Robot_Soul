# YuWen Robot Soul - Poetic Interaction Engine (Grade 1)

<img src="./assets/logo_ts.webp" width="120" alt="SHENSIST Logo">

**YuWen Robot Soul** is a premium, fairytale sci-fi educational interface designed for Grade 1 students to interact with Tang Poetry, starting with the classic 《静夜思》(Jing Ye Si).

## 🚀 Vision
To transform the traditional classroom into a celestial theater where poetry comes alive through robotic soul interaction.

## ✨ Features
- **Celestial Atmosphere**: Glowing lunar surface, starry background, and rocket launch animations.
- **Robot State Machine**: An emotionally responsive robotic entity that changes expressions based on poem context.
- **Custom Voice Audio**: High-fidelity `.wav` recordings for authentic poetic instruction.
- **AUI Branding**: Integrated Shensist Matrix AUI branding for a professional classroom experience.
- **Mobile Responsive**: Fully optimized for both wide-screen projectors and handheld devices.
